# Medical RAG System

RAG system for medical PDF documents using:

- Qdrant
- USER-bge-m3 embeddings
- PDF preprocessing
- Chunking
- Semantic retrieval

## Features

- PDF text extraction
- Cleaning and preprocessing
- Recursive chunking
- Embedding generation
- Qdrant vector search
- Retrieval testing
